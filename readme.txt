== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Accarousel Options -> Accarousel Options Page screen to configure the plugin
4. Add New Accarousel Items -> Accarousel Items page then click Add new button. 
5. Use shortcode in the post or page. Just mention the shortcode : [accarousel] 
6. If you want Accarousel in your theme php, Place the following code <code><?php do_action('Your Shortcode'); ?></code> in your templates.

Type the following shortcode in your post or page : 
<pre>[accarousel]</pre>
